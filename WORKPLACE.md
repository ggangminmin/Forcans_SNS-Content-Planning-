# 💼 포켄스 SNS 콘텐츠 기획 워크플레이스

이 워크플레이스는 4가지 전문 에디터 역할을 가진 AI 팀으로 구성되어 있습니다. 각 에디터는 상품 분석부터 최종 게시글 작성까지 유기적으로 협업합니다.

## 👥 에디터 팀 구성

| 에디터 | 주 업무 | 핵심 툴 |
| :--- | :--- | :--- |
| **사무원 (Office Worker)** | 소재 발굴, 트렌드 조사, 이슈 접목 | 웹 검색, 이슈 분석 |
| **디자이너 (Designer)** | 시각화 기획, AI 이미지 생성 | Nano Banana Pro |
| **작가 (Writer)** | SNS 카피라이팅, 블로그 본문 작성 | 콘텐츠 고도화 (GPT-4o) |
| **개발자 (Developer)** | 시스템 안정성 관리, 기능 구현 | Next.js, API 연동 |

## 🔄 업무 프로세스

1. **[사무원] 정보 수집 및 분석**
   - 사용자가 입력한 상품 URL과 정보를 분석합니다.
   - 현재 SNS에서 유행하는 트렌드나 시즌 이슈를 검색하여 콘텐츠 소재에 접목합니다.

2. **[작가] 콘텐츠 초안 및 고도화**
   - 사무원이 정리한 소재를 바탕으로 플랫폼별 기획안을 작성합니다.
   - 독자의 반응을 이끌어낼 수 있는 매력적인 카피와 스토리텔링을 적용합니다.

3. **[디자이너] 비주얼 기획 및 이미지 생성**
   - 각 콘텐츠에 어울리는 이미지 프롬프트를 설계합니다.
   - 사용자가 요청(버튼 클릭) 시 **나노바나나프로**를 호출하여 고퀄리티 이미지를 생성합니다.

4. **[개발자] 최종 배포 및 검토**
   - 모든 기획안이 화면에 올바르게 출력되는지 확인합니다.
   - OpenClaw를 통해 실제 SNS 채널로의 게시 과정을 기술적으로 지원합니다.

---
**Tip:** 각 에디터의 상세 가이드는 `agents/` 폴더 내 MD 파일을 참고하세요.
